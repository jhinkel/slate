    Introduction

    This api allows you to see data about an ad account, ads within that ad account, and posts those ads promote, all in one database.

    Data is obtained via and HTTP request using the CURL command. You can view code examples in the dark area to the right.

    All data is returned as a JSON string.

    Formatting

    #With CURL, make a call to the following URL

curl "https://www.kosocialconnections.com/AdTracker
/apiDbGet.php?endpoint=ENDPOINT_HERE&params=PARAM1,PARAM2
&email=EMAIL_HERE"

#PARAM1 and PARAM2 might or might not be used. If 
#an endpoint has no parameters, do not use the 
#"&params=PARAM1,PARAM2" section.

#also, the accountExists endpoint requires no
#email authentication!  When calling this
#endpoint, you do not have to add
#"&email=EMAIL_HERE"

    With the exception of the accountExists endpoint,

    APConnect uses email authentication to allow access to the API. You can fill out our form (FORM LINK HERE) to request that your email be added to the whitelist.

    APConnect expects for the email address to be included in all API requests to the server with an email parameter appended to the end of the CURL command as follows:

    &email=EMAIL_HERE

    when calling the accountExists endpoint, omit the &email=EMAIL_HERE part of the CURL command.

    You must replace EMAIL_HERE with the email address you used to register for API access, unless you are calling the accountExists endpoint.

    Error Description
    No values returned endpoint does not exist. Try checking spelling and/or capitalization
    type1 “Please specify an endpoint to query” - endpoint was not included in the CURL command.
    type2 “Please specify an account to lookup” - a parameter (ad account id) was not specified
    type3 “Please specify both account id and Age” - a parameter was not specified.
    type4 “Please specify a country code to lookup” - country code was not specified for an endpoint that required it.
    type5 “Please specify a gender to lookup (either male, female, or unknown).” - a gender was not specified for an endpoint that required it.
    type6 “Please specify a post id to lookup” - a parameter (post id) was not specified
    type7 “Please specify an ad id to lookup (ad ids can be obtained from the ‘adIdsFromAccount’ endpoint.” - a parameter (ad id) was not specified.
